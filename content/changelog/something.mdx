# All components

## Code

```py
print("Hello world")
print("Hello world") # [!code focus]
print("Hello world")
```

```py
print("Hello world")
print("Hello world") # [!code error]
print("Hello world") # [!code warning]
```

```java title="hello.java" showLineNumbers {3}
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}
```

```js
const colors = {
  R: "RED", // [!code ++]
  R: "red", // [!code --]
}
```

```json title="Request" /John/#r /30/#g /car/#b /null/#y 
{
  "name": "John",
  "age": 30,
  "car": null
}
```

You can then reuse the colors in your doc for instance you can say hey look at <R>John</R> he's in red, and <G>30</G> is green, and <B>car</B> is blue, and <Y>null</Y> is yellow.
